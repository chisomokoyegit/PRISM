# Example data

- **`streamlit_test_projects.json`** — **30** synthetic projects (array of objects): **10 High**, **10 Medium**, and **10 Low** `risk_level` rows, with metrics and comments aligned to each tier. Includes JIRA-style fields **`defect_rate`**, **`blocker_ratio`**, **`reopen_rate`**, and **`churn_rate`** (0–1) for MCDA / Rankings / Compare. Use **Upload Data** in the app and select this JSON. Compatible with `DataLoader` and `DataValidator` required fields.
